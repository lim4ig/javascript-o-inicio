# Javascript -  O início
> Estudos iniciais de javascript

----

## lima

- 1. Fundamentos da Linguagem:

Variáveis: Entender como declarar (var, let, const) e armazenar dados.

Tipos de Dados: Conhecer os tipos primitivos como String (texto), Number (números), e Boolean (verdadeiro/falso).

Operadores: Saber usar operadores aritméticos (+, -, *, /), de comparação (==, ===, >) e lógicos (&&, ||).
- 2. Estruturas de Controle:

Condicionais (if/else): Permitir que o código tome decisões e execute diferentes blocos com base em condições.

Loops (for, while): Usados para repetir uma tarefa várias vezes, como percorrer uma lista de itens.
- 3. Estruturas de Dados Essenciais:

Arrays: Aprender a trabalhar com listas ordenadas de dados.

Objetos: Entender como agrupar dados e funcionalidades em uma única estrutura, usando pares de chave-valor.
- 4. Funções:

Saber como criar blocos de código reutilizáveis que podem ser chamados para executar tarefas específicas.
- 5. Interação com a Página (DOM e Eventos):

Manipulação do DOM (Document Object Model): Este é o ponto crucial para o desenvolvimento web. Aprender a selecionar elementos HTML (por ID, classe, etc.) e a modificar seu conteúdo, estilo e atributos.

Eventos: Entender como responder a ações do usuário, como cliques de mouse, pressionamento de teclas e envio de formulários, para tornar a página dinâmica.
